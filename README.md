# Gift AI

## AI 맞춤 선물 추천 서비스

사용자의 관계, 나이, 취향, 예산, 상황을 분석하여 AI가 개인에게 맞는 선물을 추천해주는 서비스입니다.

## 프로젝트 소개

선물을 고를 때 상대의 취향과 상황을 고려하는 과정은 생각보다 많은 고민이 필요합니다.

Gift AI는 생성형 AI를 활용하여 사용자가 입력한 정보를 분석하고 개인의 상황에 맞는 선물과 추천 이유를 제공합니다.

## 주요 기능

### 1. 사용자 정보 입력

사용자가 입력:

- 관계
- 나이
- 취향
- 예산
- 상황

### 2. AI 선물 추천

GPT API를 활용하여:

- 맞춤 선물 추천
- 추천 이유 생성

### 3. 결과 시각화

추천 결과를 카드 형태의 UI로 제공하여 사용자가 쉽게 확인할 수 있도록 구성

## 사용 기술

- Python
- Streamlit
- OpenAI API
